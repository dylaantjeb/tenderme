# KPI/SLA Overzicht voor Onderhoud en Verduurzaming Gemeentelijke Wegen 2025–2029

In dit document worden de KPI's en SLA's beschreven die van toepassing zijn op het contract voor het onderhoud en de verduurzaming van de gemeentelijke wegen van de Gemeente Utrecht van 2025 tot 2029. Elke KPI/SLA is SMART geformuleerd en voldoet aan de gespecificeerde eisen.

| KPI/SLA | Target | Meetmethode | Frequentie | Escalatie | Verantwoordelijke | Link W-xx/criterium | Meetbron |
| ------------------------------------- | ---------------------------------- | ------------------------------------------- | -------------------- | --------------------------------------- | --------------------------------------- | --------------------- | --- |
| Klanttevredenheid | ≥ 8,0 | Enquête onder gebruikers | Jaarlijks | Bij < 7,5: Directe rapportage aan de Gemeente | Projectmanager | W-01 | CMMS / Audit / Sensor |
| Responstijd storingen | ≤ 2 uur | Logboek storingsmeldingen | Continue monitoring | Bij > 2 uur: Directe rapportage aan de Gemeente | Technical Support Lead | W-02 | CMMS / Audit / Sensor |
| Opleveringen binnen planning | ≥ 95% | Projectplanning versus opleverdata | Na elke oplevering | Bij < 90%: Evaluatie tijdens maandelijkse meeting | Projectmanager | W-03 | CMMS / Audit / Sensor |
| CO₂-reductie per project | ≥ 15% ten opzichte van baseline | CO₂-uitstootrapportage | Per project | Bij < 10%: Actieplan opstellen | Duurzaamheidscoördinator | W-04 | CMMS / Audit / Sensor |
| Kwaliteit van uitgevoerde werkzaamheden | ≥ 90% | Inspectie rapportages | Na iedere fase van de werkzaamheden | Bij < 85%: Herstelplan opstellen | Hoofd Werkvoorbereiding | W-05 | CMMS / Audit / Sensor |
| Digitalisering rapportages | 100% binnen de week | Aantal ingediende digitale rapportages | Wekelijks | Bij niet-indiening: Waarschuwing aan projectmanager | Projectmedewerker | W-06 | CMMS / Audit / Sensor |
| Tijdige communicatie | 100% binnen 24 uur | Communicatie log | Continue monitoring | Bij overschrijding: Persoonlijke opvolging | Communicatiefunctionaris | W-07 | CMMS / Audit / Sensor |
| Veiligheid bij uitvoeringen | 0 ongevallen | Veiligheidsrapportages | Maandelijks | Bij 1 ongeval: Directe evaluatie en actieplan | Veiligheidscoördinator | W-08 | CMMS / Audit / Sensor |
| Kansen op innovatieve oplossingen | ≥ 3 per jaar | Innovatie-rapportage | Jaarlijks | Bij < 2: Vergadering met stakeholders | Innovatiemanager | W-09 | CMMS / Audit / Sensor |

| Effectiviteit risicobeheersing | ≥ 80% van geïdentificeerde risico's gemitigeerd | Risicodossier updates | Kwartaal | Bij < 70%: Evaluatie met betrokken partijen | Risicomanager | W-12 | CMMS / Audit / Sensor |
